
### 주제 : 자바스크립트 로컬스토리지
---

### 메모 : 


==로컬스토리지는 함수가 아닌 객체형식으로 만들어야함==


>예시

            const localstorage = {
                setItem : (key, value)=>{
                    localstorage.setItem(key, JSON.stringify(value));
                },

                getItem : (key)=>{
                    return JSON.parse(localstorage.getItem(key));
                }
            }
